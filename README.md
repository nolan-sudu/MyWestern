# MyWestern

**A secure, student dashboard for Western University that displays various university resources, events, and course information in one customizable platform.** (IN PROGRESS)

---

## Tech Stack

- **Backend:** Node.js, Express, Prisma ORM, PostgreSQL  
- **Authentication:** Email/password login with JWT-based sessions  
- **Frontend:** React.js (Vite or Create React App)  
- **Other:** bcrypt for password hashing, cookie-parser for secure cookies, dotenv for environment variables  

---

## Features

- Secure login/signup restricted to `@uwo.ca` emails  
- Passwords hashed with bcrypt  
- JWT-based authentication stored in HttpOnly cookies  
- Clean, modular project structure with separate frontend and backend  
- Easily extendable for features like course listings, events, grades, and more
